# Planea Tu Obra

## Project Documentation
This project is a price comparison web application for construction materials, furniture, and accessories in Spain. It allows users to easily compare prices from various suppliers to find the best deals, ensuring cost-effective purchasing decisions.

## Features
- **Product Comparison**: Compare prices for various construction materials, furniture, and accessories from multiple suppliers.
- **User Reviews**: Users can leave reviews for products and suppliers.
- **Supplier Listings**: Easily find suppliers for construction materials and accessories.
- **Search Functionality**: Quickly search for specific products or categories.
- **Wishlist**: Save favorite products for future reference.

## Tech Stack
- **Frontend**: React.js, Redux, Bootstrap
- **Backend**: Node.js, Express.js
- **Database**: MongoDB
- **Hosting**: Heroku, AWS S3 (for file storage)

## Installation Instructions
1. **Clone the repository**:
   ```bash
   git clone https://github.com/elymhh/Planea-tu-obra.git
   cd Planea-tu-obra
   ```
2. **Install dependencies**:
   For the frontend:
   ```bash
   cd frontend
   npm install
   ```
   For the backend:
   ```bash
   cd backend
   npm install
   ```
3. **Set up environment variables**:
   Create a `.env` file in both the frontend and backend folders and include necessary configurations such as API keys, database URI, etc.
4. **Run the application**:
   For the frontend:
   ```bash
   npm start
   ```
   For the backend:
   ```bash
   npm start
   ```
5. **Navigate to** `http://localhost:3000` in your browser to see the application.
